# Praktik Pedagogik Deep Learning dalam Kurikulum Merdeka

Praktik pedagogik Deep Learning dalam Kurikulum Merdeka adalah pendekatan yang memfokuskan proses belajar pada kedalaman materi (materi esensial), bukan keluasan hafalan. Pendekatan ini diperkenalkan oleh Mendikdasmen Abdul Mu'ti untuk menyempurnakan Kurikulum Merdeka melalui tiga pilar utama dan metode pembelajaran inovatif. [^1][^2][^3][^4][^5]

## 1. Metode Utama dalam Deep Learning

Untuk mengimplementasikan deep learning, guru dapat menggunakan beberapa metode/model pembelajaran berikut: [^6]

| Metode                            | Deskripsi                                                                                                           |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Project-Based Learning (PjBL)** | Menempatkan siswa sebagai "peneliti kecil" yang menyelesaikan proyek nyata secara lintas disiplin ilmu.             |
| **Problem-Based Learning (PBL)**  | Menghadapkan siswa pada masalah kompleks tanpa jawaban tunggal untuk melatih pemecahan masalah secara ilmiah.       |
| **Inquiry-Based Learning**        | Mengajak siswa menemukan konsep sendiri melalui proses bertanya, penyelidikan, dan analisis data berdasarkan bukti. |
| **Discovery Learning**            | Menekankan pada proses penemuan pengetahuan melalui eksplorasi dan eksperimen mandiri.                              |
| **Cooperative Learning**          | Fokus pada kolaborasi antar siswa untuk mencapai tujuan belajar bersama.                                            |

[^6][^7][^8][^9][^10]

## 2. Tiga Pilar Pembelajaran (The Full/Mind Pillars)

Metode-metode di atas dijalankan berdasarkan tiga prinsip inti yang digagas kementerian: [^3][^11][^12]

| Pilar                   | Penjelasan                                                                                                                          |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| **Mindful Learning**    | Pembelajaran yang sadar dan aktif, di mana guru memahami keunikan tiap anak dan siswa fokus pada proses berpikir mereka.            |
| **Meaningful Learning** | Guru mengaitkan materi dengan pengalaman nyata siswa agar mereka paham tujuan dan kegunaan ilmu tersebut bagi karier dan kehidupan. |
| **Joyful Learning**     | Menciptakan interaksi yang sehat dan menggembirakan sehingga siswa termotivasi untuk terus belajar.                                 |

[^2][^13][^14]

## 3. Kerangka Kerja Keterampilan 6C

Praktik ini bertujuan membangun enam kompetensi global pada siswa: [^15][^16]

1. **Critical Thinking**: Mampu menganalisis dan memecahkan masalah.
2. **Creativity**: Menghasilkan ide atau karya inovatif.
3. **Communication**: Terampil menyampaikan gagasan secara jelas.
4. **Collaboration**: Mampu bekerja sama dalam tim.
5. **Character**: Memiliki integritas dan empati (compassion).
6. **Citizenship**: Memiliki rasa tanggung jawab sebagai warga negara/dunia.

[^7][^15][^16][^17]

## 4. Tahapan Pengalaman Belajar

Dalam praktiknya, siswa tidak langsung diuji, melainkan melalui siklus: [^18][^19]

| Tahap                        | Aktivitas                                                         |
| ---------------------------- | ----------------------------------------------------------------- |
| **Memahami (Understanding)** | Mengonstruksi pengetahuan secara aktif.                           |
| **Mengaplikasi (Applying)**  | Menggunakan pengetahuan untuk membuat karya atau solusi.          |
| **Merefleksi (Reflecting)**  | Mengevaluasi apa yang dipelajari dan dampaknya pada diri sendiri. |

[^18]
